Export Instructions

  1. Method 1 (Browser): Right-click on each logo above and select "Save image as..." to download as PNG
  2. Method 2 (Screenshot): Take a screenshot of each logo box, crop to exact size
  3. Method 3 (Design Tool): Copy the SVG code and import into Figma/Sketch/Illustrator, then export at required sizes
  4. Replace the existing files:
    • public/icon-512.png - Use the 512×512 light version
    • public/icon-192.png - Use the 192×192 light version
    • public/favicon.ico - Convert the 32×32 to .ico format
  5. For best results, ensure the background is transparent when exporting
